数据结构-表

发表于 2018-01-05 | 分类于 数据结构

关于表的一些知识

基本理论

线性表

除首节点和尾节点外，每个节点只有一个前驱，一个后继的n(n>=0)个节点的序列。

顺序表

采用顺序存储的线性表。
顺序存储:使用一组地址连续的存储单元进行存储。

链表

采用链式存储的线性表。
链式存储:结点的逻辑次序与物理次序不必一致。

散列表

查找:完成值v到存储地址d的映射f。f为散列函数。
根据哈希函数和处理冲突的方法将一组key集合映射到有限连续地址集上，并将key在地址集上的‘像’作为记录在表中的存储位置，这样的表为哈希表。

key可以为任意类型。非数字需转换为数字。
key在地址集上的‘像’为H(key)或Hi

  • 常用散列函数
    • 直接定界法
      H = a*key + b;
    • 取余法
      H = key % p
      0<模数p - 1<散列表长度m - 1
    • 提取数位法
      从key中取若干位拼成整数作为d
  • 散列冲突
    链接法
    开放地址法
    Hi = (H(key) + di) % p(i1,2…冲突次数)
    • 线性探测
      hi = i
    • 平方探测
      hi = i**2
    • 随机探测
      hi+1 = (bhi+p)%2k(b为奇数，p为偶数，k为整数，h0为0)
      h1,h2…hi为伪随机序列
